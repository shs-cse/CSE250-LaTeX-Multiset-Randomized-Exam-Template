\titledquestion{CO3}[5]
% hidden (needed for nice answers)
% different randint ranges guarantee different eq resistances
\xdef\rx{\fpeval{randint(1,3)}}
\xdef\ry{\fpeval{randint(4,6)}}
\xdef\rz{\fpeval{randint(6,9)}}
\xdef\mul{\fpeval{\rx+\ry+\rz}}
% question
\xdef\ra{\fpeval{\rx*\mul}}
\xdef\rb{\fpeval{\ry*\mul}}
\xdef\rc{\fpeval{\rz*\mul}}
% answers
\xdef\rab{\fpeval{\rc*(\ra+\rb)/(\ra+\rb+\rc)}} % \rz*(\rx+\ry)
\xdef\rbc{\fpeval{\ra*(\rb+\rc)/(\ra+\rb+\rc)}} % \rx*(\rz+\ry)
\xdef\rca{\fpeval{\rb*(\rc+\ra)/(\ra+\rb+\rc)}} % \ry*(\rz+\rx)

Which of the following statements is true?
